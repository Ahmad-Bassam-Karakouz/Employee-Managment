

🧑‍💼 نظام إدارة الموظفين - C# WinForms

# قمتُ بانشاء تطبيق سطح مكتب بسيط باستخدام C# Windows Forms، للتدرب على :
* عمليات CRUD
* التعامل مع  controls
* التعامل مع Events   
* معالجة واجهة المستخدم

## الميزات:
* إضافة موظفين إلى قائمة عرض (ListView)
* البحث عن الموظفين باستخدام رقم التعريف أو تطابق جزئي (رقم التعريف / الاسم)
* تعديل وتحديث بيانات الموظف المختار
* حذف الموظفين مع تأكيد الحذف
* التحقق الأساسي من صحة المدخلات (الحقول الإلزامية + التحقق من العمر)
* أنماط عرض متعددة لقائمة العرض (التفاصيل، القائمة، المربعات، الأيقونات)

### ما تدربت عليه:
* العمل مع عناصر تحكم Windows Forms
* التعامل مع الأحداث في لغة C#
* استخدام ListView و SubItems
* منطق التحقق الأساسي من الصحة
* تنظيم الكود في دوال لتحسين سهولة القراءة
